# Toice

Toice is a Android app currently being developed, it listens to calls in real-time and warns the user when the person on the other
side of the call is a scammer. There is a guardian system to notify a more capable person chosen on the users phone, or scale the warning to local authorities.

Watch the demo at https://toice.davidconstantindamian.com and join the email list to be notified when it launches!

## Tech stack
Kotlin

Python: Mistral, Faster-Whisper
